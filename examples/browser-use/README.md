# Browser-Use Demos

ACE framework demonstrations using browser automation.

## Demos

- **form_filler_demo/**: Learn form validation rules and achieve zero errors
- **domain_checker_demo/**: Learn optimal domain search strategies

## Quick Start

```bash
pip install ace-framework[demos]
export OPENAI_API_KEY="your-key"
cd form_filler_demo  # or domain_checker_demo
python session1_baseline.py
python session2_with_ace.py
```
